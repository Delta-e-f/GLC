## ПРОТОКОЛ ЗАПРОСОВ C ПУ К ГЛ
          
	       
| Номер байта   | Описание                              | Значение                              |
|---------------|---------------------------------------|---------------------------------------|
| 0             | Адрес устройства (ГЛ)                 | 0x26                                  |
| 1             | Режим Работы                          | 0x00 - скорость                       |
|               |                                       | 0x01 - положение                      |
| 2             | Смена режима                          | 0x00 - телеметрия                     |
|               |                                       | 0x01 - движение                       |
| 3             | Смена состояния тормозов              | 0x00 - сохранить состояние            |
|               |                                       | 0x01 - сменить состояние              |
| 14            | Скорость вращения (курс) СТБ          | 0xXX - значение в формате uint8_t     |
| 15            | Скорость вращения (курс) МЛБ          | 0xXX - значение в формате uint8_t     |
| 16            | Скорость вращения (тангаж) СТБ        | 0xXX - значение в формате uint8_t     |
| 17            | Скорость вращения (тангаж) МЛБ        | 0xXX - значение в формате uint8_t     |
| 45            | Управление заслонкой                  | 0x01 - смена состояния                |
| 127           | Адрес отправителя                     | 0xAA - 0b1010'1010                    |
